# Curse-of-the-Forbidden-Forest
Curse of the Forbidden Forest adalah game third-person survival exploration di mana pemain berperan sebagai Louis yang mencari adiknya di hutan terkutuk. Pemain menjelajah, mengumpulkan clue, dan menghadapi makhluk terkutuk dengan peluru terbatas. Pilihan pemain menentukan akhir cerita: selamat, hilang, atau tragedi.
